## 它跟番茄钟不是一回事

很多人第一反应是：这不就是番茄钟吗？

其实不太一样。番茄钟的逻辑是：你先设定一段专注时间，倒计时结束提醒你。它更像是一个纪律工具，管的是"你有没有专注"。

BalmyTime 的逻辑是另一套——你设定一个固定的休息时间点，比如每小时 50 分钟的时候，它就弹出来提醒你动一动。不管你在干什么，只要到点了，它就出现。这个思路有点像 Apple Watch 的站立提醒，给你一种绝对的时间坐标感。

好处是什么？它不需要你去启动、去点开始、去盯着倒计时，存在感极低。你就专注做自己的事，它在该出现的时候自然出现。

**定时弹窗休息提醒**
按自定义的休息时间点弹窗，专业版支持自定义休息时长，还有严格休息模式——弹出来之后真的要你停下来，不是可以随手关掉的那种。

**屏幕使用时间统计**
自动记录你用了哪些 App、用了多久，完全自动，不用手动打开什么记录模式。

**日历式作息数据回顾**
以日历形式展示你最近的工作和休息分布，能直观看出哪天坐得太久、哪天休息比较规律。

**每日心情记录**
可以用一句话记录当天的状态或感受。小功能，但一段时间之后回看，还挺有意思的。

**白噪音陪伴**
休息时段可以播放白噪音，海浪声、水滴声、猫咪呼噜声，随自己心情选。
